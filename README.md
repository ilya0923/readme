Программа будет считывать текстовую строку и удалять лишние пробелы с обоих концов.

Формат ввода
Одна строка текста с возможными лишними пробелами в начале и/или конце (например, " Python is amazing ")

Задача
Вывести входную строку с удалёнными пробелами с обоих концов.

Пример
Введите строку с пробелами в начале или конце: Hello
Результат: Hello

Ввод:Process finished with exit code 0
